  • 살펴보기

    휘캡 프로젝트는 키캡 온라인 쇼핑몰 콘셉트의 웹 사이트 입니다.
    지인의 요구사항에 맞추어 제작한 프로젝트입니다.
    Spring MVC를 사용하여 RESTful API를 구현한 웹 사이트 이며 주요기능으로 Oauth 소셜로그인, 장바구니 및 결제 기능, AJAX를 이용한 찜하기 기능, ckEditor5 API를 이용한 글 작성 및 파일업로드 기능, 인터셉터(Interceptor)를 활용한 로그인 세션 처리, CSRF 토큰을 이용한 보안강화 등을 개발하였으며 기술 스택 복기 및 실력향상 초점에 맞추어 제작하였습니다.

    사용자가 로그인 하면 Spring Security의 기능 CSRF 토큰을 발급하여 세션에 저장하여 보안을 강화하였고 인터셉터(Interceptor)로 해당 회원 및 관리자 정보가 세션에 저장되며 회원과 관리자를 구분하였습니다.
    관리자는 관리자 페이지에서 유저 및 상품 관리, 문의 답변, 상품 등록 등을 할 수 있습니다.
    회원은 로그인 및 회원가입 할 수 있으며 회원은 소셜 로그인을 통하여 별도의 회원가입 없이 로그인 할 수 있고 원하는 한개의 상품을 결제 하거나 장바구니 기능을 통한 여러 상품을 한꺼번에 결제할 수 있으며 마이페이지 기능을 통하여 나의 구매후기 및 리뷰 작성 등을 할 수 있습니다.
    AJAX를 사용하여 문의 게시글을 등록한 회원 과 로그인 한 회원이 일치해야 해당 문의 게시글을 볼 수 있습니다.

    소셜로그인은 SSL 인증서(보안 인증서)가 필요하여 코드만 구현된 상태이며 등록된 특정 계정만 이용가능 합니다.
